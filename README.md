Backend notes:
- Set FIREBASE_SERVICE_ACCOUNT to the JSON string of the service account.
- Set FIREBASE_DATABASE_URL to your RTDB URL (e.g. https://fir-u-c-students-web-default-rtdb.firebaseio.com).
- Optionally set OPENAI_API_KEY to enable AI responses.
- Deploy root to backend on Render with build `npm install` and start `npm start`.
